# AgentHub

One hub for every AI coding agent.

[中文版本](./README.CN.md)

---

## System Requirements

- Windows: Windows 10 and above
- macOS: macOS 12 (Monterey) and above
- Linux: Ubuntu 22.04+ / Debian 11+ / Fedora 34+ and other mainstream distributions

---

## Features

### Unified MCP Management

- Configure once, deploy to any of 22+ supported agents
- Stdio, SSE, and StreamableHttp transports
- Enable or disable servers without removing them
- View and audit servers across all agents in one command

### Skills Market + Internal Git

- Import `.skill` packages or author skills with SKILL.md frontmatter
- Browse and install from the skills.sh marketplace
- Search and install from your internal Git skills repository
- Configure the internal skills repository URL in `Settings > Integrations`

### MCP Market (Official + Internal)

- Search the official MCP Registry and internal MCP catalog repositories
- Configure the internal MCP catalog repository URL in `Settings > Integrations`
- Keep transport metadata consistent across public and internal sources

### Integrations and Credentials

- Choose your preferred code editor for opening skill folders/files
- Save and manage credentials (for example, GitHub access tokens)
- Global and project scopes for MCP, skills, and sub-agents
